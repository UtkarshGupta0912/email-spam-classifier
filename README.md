# Email/SMS Spam Classifier

Machine Learning project that classifies SMS/Email messages as Spam or Not Spam.

## Technologies Used
- Python
- Scikit-Learn
- TF-IDF Vectorizer
- Multinomial Naive Bayes
- Streamlit
